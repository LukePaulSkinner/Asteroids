# Asteroids

## Controls

| Key  | Action |
| ------------- | ------------- |
| Left arrow key  | Turns the ship left  |
| Right arrow key  | Turns the ship right  |
| Up arrow key  | Applies thrust to the ship  |
| Spacebar  | Fires a projectile from the ship  |
| B key | Opens the shop while the ship is near it   |
